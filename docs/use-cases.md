# Use Cases

## 1) Build a Tofu-Free Study Set

1. Open Setup (`Shift+S`) or Advanced (`Shift+R`).
2. Enable font filter and specify the actual render font family/path.
3. Rebuild DB.
4. Confirm build summary shows active font coverage and `excluded_font` count.
5. Relaunch in same render font environment.

## 2) Compare JP/CN Readings for One Glyph Family

1. Search for a seed glyph (`/`).
2. Keep JP and CN panels visible (`1`, `2`).
3. Use `Up/Down` to step related glyphs from shown JP/CN word rows.
4. Use `Shift+Left/Shift+Right` to choose alternate glyphs in multi-glyph words.
5. Press `Enter` to jump.

## 3) Explore Radical Neighborhoods Under Current Filters

1. Apply desired filters (`f`) and quick no-reading mode (`N`) if needed.
2. Open radicals (`r`).
3. Pick radical and adjust stroke subset (`[`/`]` in TUI; buttons in GUI).
4. Jump candidates with `Enter`.

## 4) Build a Personal Review Queue

1. Bookmark useful glyphs with `b`.
2. Add per-glyph notes (`n`) and global strategy notes (`g`).
3. Open bookmark list (`B`) to navigate/jump and prune (`x`).
4. Save filter presets for recurring review slices.

## 5) Investigate Shape and Sound Relations

1. Toggle phonetic overlay (`s`) and components (`c`) as needed.
2. Use `Up/Down` to select phonetic row members.
3. Jump with `Enter`.
4. Compare variants via Variants panel (`4`, then `Tab` focus).
